# Installation

Follow these steps to install the package.

## Prerequisites

You need Node.js 18+ and a package manager.

## Install

Run the install command for your preferred package manager.

### Using npm

```bash
npm install pagefind @pagefind/md-indexer
```

### Using bun

```bash
bun add pagefind @pagefind/md-indexer
```

## Configuration

After installation, configure the indexer in your build script.
